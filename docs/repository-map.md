# Repository Map

See [TOC1](../../TOC1/) for the high-level table. This page explains the rule of thumb:

- Store instructions in `agents`.
- Store textual debugging knowledge in `memory`.
- Store visual/manual/media rendering in `web`.
- Store action/API behavior in `api`.
- Store organization profile content in `.github`.
- Store the public documentation portal in `wiki`.
- Store raw intake in `scanner`.
- Store reviewed facts in `canonical`.
